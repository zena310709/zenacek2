# Training Planner

## Автор
Чекмарева Евгения

## Описание
Приложение для планирования тренировок с возможностью фильтрации по типу и дате. Данные сохраняются в формате JSON.

## Примеры использования
1. Запустите приложение.
2. Введите данные тренировки (дату, тип, длительность) и нажмите "Добавить тренировку".
3. Используйте поля фильтрации для поиска записей по типу и дате.
